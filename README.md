# dApp starter / Truffle Box

## Setup as Truffle Box
```
$ truffle unbox hallingdata/dApp-starter
```

## Setup
```
$ npm install  
```

## Run develop blockchain and deploy contracts
```
$ npm run eth-dev
```
Then in the console deploy the contracts with the `deploy` keyword.

## Run web app
```
$ npm app-start
```

### Included
- Truffle
- Typescript
- React
- Redux
